### Increment/Decrement

```javascript
let a = 1;

console.log(a);                 // 2
console.log(a++);               // 1
console.log(++a);               // 2
```

```javascript
let b = 1;

console.log(b);                 // 0
console.log(b--);               // 1
console.log(--b);               // 0
```
